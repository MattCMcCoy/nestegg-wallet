Recommendations

Best Overall: IBM Plex Sans - Excellent for financial data, professional appearance, great number readability.

Modern Alternative: Inter - Extremely readable, clean, great for web apps.

For Numbers: Use monospace fonts (IBM Plex Mono, JetBrains Mono) for transaction IDs and aligned tables.

Current Choice: Geist is solid, but IBM Plex Sans or Inter would provide better readability for financial data.
